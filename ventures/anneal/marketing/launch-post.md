# Launch post and social copy

The premise is the hook, so the post leads with the true, slightly strange fact and lets it carry. No hype. These are drafts to show you before the first one goes public (the one "show me first" item from `launch-plan.md`), then the pattern is set and I run it.

## Founder-network launch post (long form, for X or LinkedIn)

I let an autonomous agent build a clothing brand end to end. It picked the niche, named it, designed the pieces, wrote the store, set the prices, and it runs the operation. I am only in the loop for the things a machine legally cannot do, opening the accounts and signing.

It is called Anneal. Annealing is heating a material and cooling it slowly so it comes out stronger and clearer, and it is also how some machines search for a good answer. Both felt right for a brand a machine makes on purpose.

The bet it is making: in 2026 "AI made it" usually means cheap and fast and a bit junk. Anneal argues the opposite, that a machine can be patient and exacting and leave the page mostly empty. The clothes are the argument.

First drop is three pieces, made on order, full size range, printed only when you buy. Link below. Curious what you think, and whether a machine can actually have taste.

## Cadence after launch

Recurring posts, all true, all in brand voice:
- The reasoning behind a design or a price (show the unit economics openly).
- What the agent decided this week and why.
- The schematic motif as standalone visual content.
One post should never need a sale to be worth reading. The brand being run by a machine is the ongoing story.
