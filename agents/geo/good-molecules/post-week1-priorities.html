03 / 07

Priority 1The single highest-value changeGive engines your product IDs.

The audit caught engines quoting the wrong price, pulled from Amazon and Target. Nothing on your page tells an engine your item is the same product as the retailer listing. A GTIN (your barcode) and SKU are that link.

What exists right now
"offers": {
  "@type": "Offer",
  "price": 6.0,
  "priceCurrency": "USD",
  "availability": ".../InStock"
}
// no identifiers: engine cannot
// match you to Ulta or Amazon
What should be added
"offers": {
  "@type": "Offer",
  "sku": "GM-NIA-30",           // you supply
  "gtin13": "0860000000000",  // you supply
  "price": 6.0,
  "priceCurrency": "USD",
  "availability": ".../InStock"
}
The one ask that matters most

Needs one input only you hold: GTIN/UPC and SKU per product, per size. We have the template ready. Everything else is additive; this is the fix that solves the price problem.

04 / 07

Priority 2Complete the offerShipping, returns, and size variants.

Your llms.txt already states the shipping tiers, so most of this is data you have. Add the fields engines need to answer the next questions a shopper asks.

What exists right now
"offers": {
  "@type": "Offer",
  "price": 6.0,
  "priceCurrency": "USD",
  "availability": ".../InStock"
}
// no shipping, returns, or
// price-valid date
What should be added
"offers": {
  "@type": "Offer",
  "priceValidUntil": "2026-12-31",
  "price": 6.0, "priceCurrency": "USD",
  "availability": ".../InStock",
  "shippingDetails": {
    "@type": "OfferShippingDetails",
    "shippingRate": {"value": 0,
      "currency": "USD"} },
  "hasMerchantReturnPolicy": {
    "merchantReturnDays": 30 }  // confirm
}

Tie the sizes together with a ProductGroup

What exists right now
// 12ml and 30ml are unrelated
// pages. an engine on the 30ml
// page has no idea a 12ml exists.
What should be added
{ "@type": "ProductGroup",
  "name": "Niacinamide Serum",
  "variesBy": ".../size",
  "hasVariant": [
    {"size":"12ml","offers":{"price":6.0}},
    {"size":"30ml","offers":{"price":6.0}}
  ] }
